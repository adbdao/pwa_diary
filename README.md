# 《我的網頁日記》的原始碼 diary for PWA  
  
### 直接在網頁上寫日記，不必安裝APP，就不會有廣告。之前用了幾個寫日記的APP，廣告都很多。我不想花錢買APP，網路上找不到又好用又免費的APP。乾脆自己寫PWA來寫日記  
>### 製作中途發生很多問題，後來用Google的AI來製作。果然快很多，又多了：月曆、搜尋…等新功能。  
---  
> 展示網址：  https://adbdao.github.io/pwa_diary/index.html  
>###  漸進式網站PWA  
  1. Android系統上，會自動跳出〔安裝應用程式〕對話方塊。或者手動按〔點此安裝APP〕按鈕  
  2. iPhone系統上，請在瀏覽器中，使用〔加到主頁面〕
